# Earnings-Calendar-Data-Tool-
Python tool that retrieves upcoming US earnings announcements from Yahoo Finance (using chrome driver) and exports them to a formatted Excel workbook. Built using pandas, yfinance, and openpyxl to streamline earnings tracking and support trade planning.

## Features
- Retrieve upcoming US earnings announcements using chromedriver (bot)
- Filter earnings by a user-defined date range
- Export results to a formatted Excel workbook
- Group earnings by date and ticker for easy navigation
- Automatically organise data for further analysis

## Built With
- Python
- pandas
- yfinance
- openpyxl

## Why I Built This
I wanted a simple way to quickly view upcoming earnings announcements without manually searching through multiple websites. This project also gave me practical experience working with APIs, data processing, and exporting structured datasets to Excel.
